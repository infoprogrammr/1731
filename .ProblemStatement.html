Objective

Implement search functionality on the given Search box

Problem Definition

You are given a webapp that displays a list of houses from the Game of Thrones series.

The house names are clickable too. (Please try it out).

Your mission is the following:

Add search functionality:

The current Search box does not work. When you type in a house name and press Enter, it should show filtered results accordingly.

Eg: if user types in "Algood" and presses Enter, it should show him/her only those houses having the words "Algood" in their title.
